Stereo Test Sound File

Replace this file with "stereo-test-sound.mp3".

Recommended audio file specifications:
- Format: MP3
- Sample Rate: 44.1 kHz  
- Channels: Stereo (2 channels)
- Bit Rate: 128-320 kbps
- Duration: 2-5 seconds
- Content: Test tone (440Hz) or stereo test signal

Test sound example:
- Left channel: 440Hz sine wave
- Right channel: 880Hz sine wave 
- Or mono 440Hz test tone

The app will automatically use the tone generator if no sound file is present.
